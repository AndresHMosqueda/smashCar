Smash Car
Taking inspiration from Mario Kart®, Smash Car has been considred one the best local multiplayer experiences gaming can offer!


How to Play? It's super easy.

Instructions

▶️PLAYER 1
Keyboard W = Up
Keyboard A = Left
Keyboard S = Down
Keyboard D = Right
Spacebar = Shoot

▶️PLAYER 2
Arrow Up
Arrow Down
Arrow Left
Arrow Right
Keyboard L = Shoot
